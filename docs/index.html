DeFi Lending Aggregator

Real-time lending and borrowing rate aggregation across DeFi protocols. Compare rates, find opportunities, and optimize your yield.

Overview

The DeFi Lending Aggregator tracks rates from 8+ lending protocols across 5 blockchains, providing real-time data for informed DeFi decisions.

Supported Protocols

Aave V3 Compound V3 Morpho Blue Spark Benqi Radiant Fluid Moonwell

Supported Chains

Ethereum Arbitrum Polygon Base Avalanche

Features

  • Real-time rate aggregation from 8+ protocols
  • Multi-chain support (5 blockchains)
  • Yield optimization with risk-adjusted scoring
  • Customizable alerts for rate changes
  • Historical rate charts and analytics
  • Protocol risk assessment
  • RESTful API with comprehensive documentation

Quick API Reference

GET /api/v1/rates          - All lending rates
GET /api/v1/protocols       - Protocol metadata
GET /api/v1/opportunities   - Yield opportunities
GET /api/v1/history/{p}/{c}/{a} - Rate history

See full API documentation for details.
